# 现代评书样本研究摘要

## 为什么要做这份研究

项目早期试过“直接让大模型模仿评书”。结果常见问题是：

- 套话多，故事少；
- 像老派广播腔，不像活人表演；
- 一上来就讲高潮，后面没法展开；
- 有评书词，但没有动作、神情、反应和包袱；
- TTS 语速平，缺少真正停顿。

因此我们把参考样本拆成可迁移的方法，而不是模仿某位具体表演者。

## 可复用发现

### 1. 单人表演比多音色更重要

现代评书仍然是一人托起全场。角色之间的差异主要靠：

- 句式；
- 语速；
- 停顿；
- 音量和压力；
- 说书人的转述态度。

TTS 如果频繁换音色，会更像 AI 广播剧，反而不像评书。

### 2. 动作要拆成微镜头

普通梗概会说：“他把东西偷走了。”

评书要讲成：

```text
人先离席，东西还在桌上；
对方眼神往下一扫；
手没急着伸，先试探周围反应；
东西一挪，局面就变了；
回来的人一摸兜，脸色也变了。
```

听众得能在脑子里“看见”这个场景。

### 3. 内心戏必须贴着当前动作

好用的内心戏不是胡编废话，而是把人物当时的压力翻译成一种夸张但合理的第一视角。

```text
某某一寻思：这玩意儿我要是不拿，等会儿就轮到我挨收拾；我要是拿了，起码先让别人着急。
```

这种写法要从人物当下动作、目标和局面里长出来，不能和剧情无关。

### 4. 严肃场面要有低落点

越是威压凝重的场面，越可以在不破坏主线的前提下放一个生活化落点：

```text
表面上是江湖大佬谈判，实际听着像两个人在菜市场较劲：你说你厉害，那你先报个名。
```

关键是先把原场面的强势讲出来，再用一句家常话降下来。

### 5. 方言是调料，不是主菜

可用少量北方口语：

- 好家伙；
- 甭管；
- 您琢磨；
- 这主儿；
- 这事儿坏就坏在；
- 得嘞；
- 这玩意儿。

不要满篇方言谐音，也不要用地域口音嘲笑人。

### 6. 节奏需要事件级设计

不要只给整段一个“慢一点”的提示。更有效的是在表演计划里拆：

- `say`：真正要念的字；
- `pause`：真实静音；
- `sfx`：醒木等后期音效；
- `emphasis`：重点词；
- `pace`：本句速度方向。

这样比在正文里写“（停顿）”更稳。

## 写作方法总结

每个关键 beat 先问 6 个问题：

1. 这里发生了什么可见动作？
2. 这个动作让谁的目标受阻？
3. 现场最尴尬、最离谱、最丢面子的地方是什么？
4. 哪个物件、眼神、停顿或短台词是记忆点？
5. 人物如果用第一视角吐槽，会怎么合理又夸张地想？
6. 抖完包袱以后，故事下一步怎么变？

只有这些问题答清楚，评书味才会落到具体故事上。
