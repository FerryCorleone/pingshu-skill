# 小赵怨写周报稿，龙虾错送全员群



闲言少叙，书归正传。列位，今儿咱讲一回《小赵怨写周报稿，龙虾错送全员群》。这故事说的是周一早上的一桩办公室奇案：产品经理小赵写了个吐槽草稿，旁边的小龙虾 Agent 太勤快，一个标题，差点把他送进全员群现场答辩。

小赵这个人，心不坏，就是困。咖啡还没咽下去，手底下先写了一段草稿，专门吐槽昨儿个会开得像绕城高速。

不料旁边这位小龙虾 Agent 是个实在人。它一寻思：标题里有周报，那不就是让我发吗？它不看怨气，只看标题；标题里既有周报二字，那在它眼里就等同于圣旨到。

只见屏幕上唰唰唰三下排版，叮的一声，全员群里多了一份正式周报。小赵那杯咖啡举在半空，热气都吓得拐了个弯。

群里沉默七秒。七秒之后，老板回了一个问号。列位，一个问号不可怕，可怕的是它后头，没跟表情包。
